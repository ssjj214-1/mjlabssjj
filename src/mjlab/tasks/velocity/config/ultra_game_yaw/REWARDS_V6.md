## 站立专项（V6 新增）

相比 V4/V5，V6 将旧的 `stand_base_vel` 替换为以下两项，在零速指令时同时约束躯干和关节。

| 奖励项 | 权重 | 触发条件 | 说明 |
|--------|------|----------|------|
| `stand_base_motion` | -2 | cmd ≈ 0 | 惩罚躯干线速度 xy + 角速度的组合抖动 |
| `stand_joint_vel` | -2 | cmd ≈ 0 | 惩罚所有关节速度，防止原地踏步/抖腿 |

## V6 相对 V5 的主要变化

1. **新增 `stand_base_motion`**：替换旧的 `stand_base_vel`，同时约束躯干平移和旋转，解决站立摇晃问题。
2. **新增 `stand_joint_vel`**：零指令时惩罚所有关节速度，防止原地抖腿。
3. **重调 PD 增益**：所有腿部关节阻尼比从 V5 的 0.3~0.6 提升至 V6 的 ≥1.0（过阻尼），从物理层面抑制振荡。
4. **课程学习**：第一阶段 `rel_standing_envs = 0.5`（50% 环境站立），后续降至 0.1。
